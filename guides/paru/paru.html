how to install AUR helper PARU on Arch linux

In this guide or tutorial I will show you how to install AUR helper paru on arch linux

STEP1:

First Install git, to install git open terminal and type

$sudo pacman -S git

STEP2:

Install the necessary files for aur helper using:

$sudo pacman -S --needed base-devel

STEP3:

clone the paru git from this link using:

$git clone https://aur.archlinux.org/paru.git

STEP4:

After it is cloned,go to the paru directory using:

$cd paru

STEP5:

After changing the directory build your package using:

makepkg -si

STEP6:

After the package is build you can search packages on the aur repository with

$paru -Ss <package_name>

then to install the package use

$paru -S <package_name>

Note:It is advised to download packages on your own risk as it can contain harmul code!!!

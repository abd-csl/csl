how to install zsh theme on linux

In this guide or tutorial I will show you how to install zsh theme using ohmyzsh on linux

STEP1:

In your browser type ohmyzsh open the github page

STEP2:

copy any one of these links

STEP3.1:

On debian install git using

$sudo apt install git

STEP3.2:

On Arch install git using

$sudo pacman -S git

STEP4:

After installation paste the link you just copied using

ctrl+shift+V

STEP5:

After the installation type

$nano .zshrc

then find the ZSH_THEME variable and edit it to agnoster (i already changed it) Use ctrl + s and ctrl + x to save and exit

STEP6:

now to see the theme change type

exec zsh

or just reopen the terminal
